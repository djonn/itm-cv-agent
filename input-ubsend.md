Customer: UBsend A/S
Type: Java Microservices in Kubernetes & legacy-platform in .NET

## Description

UBsend is broker between online stores and freight companies with the purpose of being able to get a better price or being able to deliver the packages faster as well as being a single integration instead of many. The product consists of an Angular frontend and a microservice based backend written in Java using MongoDB databases and deployed using Kubernetes.

## My function

I became a part of a mainly new team taking over a large codebase. I optimised and built new microservices with the responsibility of integrations to external partners. In addition I was involved in solving critical problems as they arose including an Out-of-Memory problem that I was able to limit to 1/13,250 of the previous memory usage.

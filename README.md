# CV Skrive Agent

Hjælper dig med at skrive enkelte projekterfaringer.

## Sådan bruges agenten

Kald /resume-assistant skill'en med lidt information omkring kunden og projektet, evt bare løse tanker.
Hvis du prøver at forbedre en eksisterende projekterfaring kan du tilføje denne og blot referere til den.

Efter du har skrevet en projekterfaring kan du starte en ny session og kalde assistenten på ny.

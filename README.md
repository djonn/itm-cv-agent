# CV Skrive Agent

Hjælper dig med at skrive enkelte projekterfaringer.

## Sådan bruges agenten

Brug resume-interviewer agenten med lidt information omkring kunden og projektet så den har et sted at starte, evt bare løse tanker.
Hvis du prøver at forbedre en eksisterende projekterfaring kan du tilføje denne og blot referere til den.

Efter du har skrevet en projekterfaring kan du starte en ny session og kalde assistenten på ny uden det overskriver dit tidligere arbejde.

## Start program

- Start database i docker med `docker compose up -d`
- Kør `mix setup` for at installere og klargøre dependencies
- Start applikationne med `iex -S mix phx.server`

Nu kan du besøge [`localhost:4000`](http://localhost:4000) fra din browswer.
